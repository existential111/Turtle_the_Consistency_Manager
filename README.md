# Turtle, the Consistency Manager

This is a planner, specifically created for the purpose of managing dance homework/assigned practice.

---

Either visit the Github Page (bit.ly/Turtle-Consistency-Manager) or download the .html file in order to run the planner. Functionality on mobile devices is untested, and slight visual errors, bugs, or inconsistencies may occur. All data is stored locally on the browser, and data is not transferred between published and local instances.

> ***To download, go to the planner file on Github and select "Download raw file" on the top right. The HTML file should download to your device. Open it using your desired web browser. Alternatively, you can download theentire zip file through Github and unzip the entire folder.***

There are 6 options within the website; for more details, please view the User Guide either on Github or through bit.ly/Turtle-User-Guide. The program was not designed to automatically remove assignments past due date, and any assignments past due may encounter unexpected errors. 

Logging time is necessary for the program to determine the relative importance of tasks. Planning your day automatically logs time for those practices. The time within the planner is based on UTC; times may not be accurate based on your time zone. Daily assignments have their logged time reset daily at midnight, and weekly assignments at midnight on Monday every week.

Please enter values in whole minutes. 

Rename the downloaded file as necessary.  

Old files are included within the "Archived" folder.
